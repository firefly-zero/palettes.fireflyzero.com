# palettes.fireflyzero.com

Color palettes for [Firefly Zero](https://fireflyzero.com/). You can use any 16-color color palettes with Firefly Zero. To make your life easier, we've generated code snippets for different languages by Firefly Zero for top palettes from [Lospec](https://lospec.com/palette-list).

[palettes.fireflyzero.com](https://palettes.fireflyzero.com/)
